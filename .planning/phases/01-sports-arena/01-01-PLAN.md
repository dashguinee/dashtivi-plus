# Plan 01-01: ESPN Data Service

## Objective
Build `src/services/sports-data.ts` — a typed client for ESPN public API + TheSportsDB. Fetches fixtures, standings, team info with caching.

## Context
- ESPN API: `site.api.espn.com/apis/site/v2/sports/{sport}/{league}/{resource}` — no auth needed
- TheSportsDB: `thesportsdb.com/api/v1/json/3/{endpoint}` — free key `3`
- Team logos: `a.espncdn.com/i/teamlogos/soccer/500/{id}.png`
- DashTivi+ proxy: `https://stream.zionsynapse.online` (CORS-safe)
- App uses localStorage caching pattern (see @src/lib/xtream.ts for pattern)

## Tasks

### Task 1: Types + League Config
- **type**: create
- **file**: `src/services/sports-data.ts`
- **action**: Define TypeScript interfaces and league config:
  ```
  League: { id, espnSlug, sport, name, shortName, logo, country }
  Fixture: { id, date, status, homeTeam, awayTeam, homeScore, awayScore, venue, broadcast, league }
  Team: { id, name, shortName, logo, league }
  Standing: { position, team, played, won, drawn, lost, gf, ga, gd, points }
  ```
  Define LEAGUES array: Premier League (soccer/eng.1), La Liga (soccer/esp.1), Ligue 1 (soccer/fra.1), Champions League (soccer/uefa.champions), Serie A (soccer/ita.1), Bundesliga (soccer/ger.1), NBA (basketball/nba), UFC (mma/ufc)
- **verify**: Types compile with `npm run build`

### Task 2: Fetch Functions + Cache
- **type**: create (append to same file)
- **file**: `src/services/sports-data.ts`
- **action**: Implement:
  - `fetchFixtures(league: League): Promise<Fixture[]>` — GET `site.api.espn.com/apis/site/v2/sports/${league.sport}/${league.espnSlug}/scoreboard` → parse events into Fixture[]. Cache 15min in localStorage.
  - `fetchStandings(league: League): Promise<Standing[]>` — GET same base + `/standings` → parse into Standing[]. Cache 1hr.
  - `fetchTeamLogo(teamId: string, sport: string): string` — return ESPN CDN URL, no fetch needed.
  - `getLeagueById(id: string): League | undefined` — lookup helper.
  - Parse ESPN response: events[].competitions[].competitors[] has team.id, team.displayName, team.logo, score.value. events[].status.type.description has "Scheduled"/"In Progress"/"Final".
  - Error handling: AbortSignal.timeout(8000), catch returns empty arrays, console.warn.
- **verify**: `npm run build` passes. Manually test in browser console: `import('/src/services/sports-data').then(m => m.fetchFixtures(m.LEAGUES[0]).then(console.log))`
