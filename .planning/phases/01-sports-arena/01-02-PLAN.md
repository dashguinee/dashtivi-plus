# Plan 01-02: Match Day Cards + League Selector

## Objective
Build the two hero UI components for the Sports experience: league selector pills and match day fixture cards.

## Context
- @src/services/sports-data.ts exists with types + fetch functions (from 01-01)
- DashTivi+ design: dark bg (#0A0A0A), purple primary (#9D4EDD), orange sports accent (#F97316)
- Card system: `.card-press`, `.card-surface` classes exist in globals.css
- Animation: `cubic-bezier(0.16, 1, 0.3, 1)` deceleration curve, stagger 120ms
- Haptics: `tap()` for selection, `click()` for opening

## Tasks

### Task 1: LeagueSelector Component
- **type**: create
- **file**: `src/components/sports/LeagueSelector.tsx`
- **action**: Horizontal scrollable pill row of leagues.
  - Each pill: league logo (24px) + short name, orange accent when active
  - Active pill: `bg-orange-500/15 border-orange-500/40 text-orange-400`
  - Inactive: `bg-white/[0.04] border-white/[0.06] text-white/50`
  - `onSelect(league)` callback
  - Scroll container uses `scrollbar-hide scroll-fade` classes
  - `data-haptic="lush"` on scroll container
  - League logos: use `league.logo` field (ESPN CDN URL from config)
  - Props: `{ leagues: League[], activeId: string, onSelect: (id: string) => void }`
- **verify**: Renders 8 pills, horizontal scroll, tap fires onSelect

### Task 2: MatchDayCards Component
- **type**: create
- **file**: `src/components/sports/MatchDayCards.tsx`
- **action**: Horizontal scroll row of upcoming/live fixture cards.
  - Card width: 260px, rounded-2xl, card-press class
  - Layout per card:
    ```
    [League badge tiny]  [Date/Time or "LIVE" badge]
    [Home crest 40px]  vs  [Away crest 40px]
    [Home name]            [Away name]
    [Score if in-progress or final]
    [Venue tiny text]
    ```
  - LIVE matches: pulsing orange dot + "LIVE" badge, orange border glow
  - Scheduled: show kickoff time in user's timezone (`toLocaleTimeString`)
  - Final: show score, dimmed styling
  - Team crests: `team.logo` (ESPN CDN)
  - Max 15 fixtures shown, sorted: LIVE first, then by date ascending
  - Stagger animation: `vee-card-in 0.9s ... ${i * 100}ms`
  - Empty state: "No upcoming matches" with subtle text
  - Props: `{ fixtures: Fixture[], isLoading: boolean }`
- **verify**: Renders cards with team logos, scores, time. LIVE badge pulses.
