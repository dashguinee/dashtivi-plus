# Plan 01-03: Standings Widget

## Objective
Build a compact league standings table widget for the Sports experience.

## Context
- @src/services/sports-data.ts has `fetchStandings(league): Promise<Standing[]>`
- Standing: { position, team: { name, shortName, logo }, played, won, drawn, lost, gf, ga, gd, points }
- Design: dark, compact, fits in a side panel or below match cards
- This is not a full page — it's a widget that shows top 6-8 teams with expand option

## Tasks

### Task 1: StandingsWidget Component
- **type**: create
- **file**: `src/components/sports/StandingsWidget.tsx`
- **action**: Compact league table.
  - Header: league name + logo
  - Table: Pos | Team (logo 16px + name) | P | W | D | L | Pts
  - Show top 6 by default, "Show full table" expands to all
  - Active league's accent color on header
  - Row styling: top 4 = subtle green left border (UCL spots), bottom 3 = subtle red (relegation)
  - Team logos: 16px rounded, from team.logo
  - Typography: monospace for numbers, `text-[11px]`
  - Responsive: full width on mobile, max-w-sm on desktop
  - Props: `{ standings: Standing[], leagueName: string, leagueLogo: string, isLoading: boolean }`
  - Loading: skeleton pulse rows
  - Empty: "Standings not available" text
- **verify**: Renders table with team logos, correct position badges

### Task 2: Sports Data Hook
- **type**: create
- **file**: `src/hooks/useSportsData.ts`
- **action**: React hook that manages sports data state.
  ```typescript
  function useSportsData() {
    const [activeLeague, setActiveLeague] = useState(LEAGUES[0].id);
    const [fixtures, setFixtures] = useState<Fixture[]>([]);
    const [standings, setStandings] = useState<Standing[]>([]);
    const [loading, setLoading] = useState(true);
    
    // Fetch on league change
    useEffect(() => {
      const league = getLeagueById(activeLeague);
      if (!league) return;
      setLoading(true);
      Promise.all([
        fetchFixtures(league),
        fetchStandings(league)
      ]).then(([f, s]) => {
        setFixtures(f);
        setStandings(s);
      }).finally(() => setLoading(false));
    }, [activeLeague]);
    
    return { activeLeague, setActiveLeague, fixtures, standings, loading, leagues: LEAGUES };
  }
  ```
  - Manages loading state, active league selection, data fetching
  - Returns everything the Sports experience page needs
- **verify**: Hook compiles, switches leagues, fetches data
